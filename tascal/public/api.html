APIについて

暫定的に作ったものなので、フォーマットやパラメーターなど要件に応じて変更していきます。 AIR/iPhone/Androidなどのクライアントからアクセスできるように、今のところAPIのフォーマットとしては一番お手軽なJSONのみサポートしています。

ユーザー認証

すべてのリクエストには、ユーザーの認証情報として、ユーザーごとのapiキーをパラメーターとして含める必要があります。apiキーは見ればわかるという話もありますが、 問い合わせてもらえれば教えます。

Tascalについて

TaskをCalc(ulate)して"助かる(Tascal)"みたいな感じ？ コラボレーションとプロジェクト管理をお手伝いします。

主な機能

実装する予定の主な機能は下記です。

  • メンバー同士のチャット(AIRクライアント/iphone/Android)
  • タスクのスケジューリングとレポート機能(他のメンバー間のタスク/スケジュール参照)
  • 空き時間検索と打ち合わせサジェスト
  • 過去に同じことやった人をご紹介(DRY)
  • その他、よだれがでそうな機能たくさん
